	二、一般流程控制
		1、用肯定句式代替否定句式，因为否定句式比肯定句式难以理解；
		2、用true和false做bool判断而不是用数字，因为数字会写错，但是true和false写错的概率就很小了；
		3、变量与常亮做逻辑比较时将常量放在逻辑判断符号左边，变量放在右边会有少输入等号而变成赋值运算，c系列语言特点；
		4、空语句一定要突出展示，比如分号单行显示，并用大括号封装成一个语句块；
